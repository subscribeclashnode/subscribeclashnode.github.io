Linux用户享受最大的灵活性： git clone https://github.com/xiaoming/xiaoming.git cd xiaoming ./autogen.sh ./configure --prefix=/usr --sysconfdir=/etc make -j$(nproc) sudo make install 编译选项可根据需要调整，如--with-v2ray启用V2Ray支持。

高级配置技巧

超越基础配置，小明还支持： - 路由规则自定义：通过编辑rules.json实现精细化的流量控制 - 多服务器负载均衡：在config.ini中配置多个节点实现自动切换 - 流量伪装：启用TLS/WebSocket等高级特性绕过深度包检测 - DNS泄漏防护：配置纯IPv6 DNS服务器增强隐私保护

这些功能需要通过编辑配置文件实现，建议在修改前备份原始文件。

安全使用的最佳实践

科学上网是把双刃剑，安全使用至关重要： 1. 节点选择：优先选择自建节点或信誉良好的付费服务 2. 传输加密：始终启用AEAD加密算法如chacha20-ietf-poly1305 3. 流量混淆：在严格审查环境下启用插件如simple-obfs 4. 防火墙设置：配置系统防火墙仅允许小明访问网络 5. 日志管理：定期清理连接日志，避免敏感信息留存

疑难排错指南

当遇到连接问题时，可依次检查： 1. 本地网络是否正常 2. 系统时间是否准确（误差超过2分钟会导致TLS失败） 3. 防火墙/杀毒软件是否拦截 4. 配置文件语法是否正确（可使用xiaoming -test测试） 5. 服务端状态是否正常

通过-v参数启用详细日志，往往能快速定位问题根源。
