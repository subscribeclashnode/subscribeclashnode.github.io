1. 下载推荐客户端

  • 解压后运行客户端程序；

  • 第一次启动可能会提示缺少依赖库，按提示安装即可。

3. 导入订阅链接

以V2RayN为例：

  1. 打开软件 → 找到“订阅设置”；

  4. 选择一个节点 → 点击“启用代理”即可连接。

4. 设置系统代理

部分客户端（如V2RayN）支持自动设置系统代理，但你也可以手动设置浏览器：

  • 代理地址：127.0.0.1

  • 端口号：1080（SOCKS）或1087（HTTP）

浏览器插件如SwitchyOmega也可辅助切换代理模式。

七、常见问题与解决方案

Q1：连接不上服务器怎么办？

  • 检查网络是否正常；

  • 确保订阅链接未失效；

  • 尝试更换其他节点；

  • 客户端是否被防火墙拦截。

Q2：延迟高、速度慢？

  • 选择延迟更低的节点；

  • 避开晚高峰（通常20:00-23:00）；

  • 使用分流策略，仅代理海外网站。

Q3：无法访问Netflix等服务？

  • 并非所有节点都支持流媒体解锁；

  • 火星云通常会标注支持Netflix的节点；

  • 尝试更换指定的“流媒体解锁”节点。

Q4：订阅链接更新后失效？

  • 检查是否误删订阅前缀或尾部；

  • 重新登录火星云后台，复制最新链接；

  • 检查客户端版本，旧版本可能不兼容。

FAQ

Sing-box 如何实现流量精细分流？
通过定义 routing 规则和策略组，Sing-box 可以根据域名、IP、端口或协议类型精细分流流量。例如视频走高速节点，网页走备用节点，实现流量优化和网络加速。
Mellow TUN 模式对游戏流量有何作用？
TUN 模式拦截系统 IP 层流量，实现全局代理。游戏流量通过低延迟节点传输，减少丢包和延迟，提高游戏体验的流畅性和网络稳定性。
Trojan 的证书配置错误会导致什么问题？
如果 TLS 证书无效或域名不匹配，Trojan 客户端将无法建立安全连接，导致节点无法使用。需要确保证书有效、域名正确，才能保证正常访问。
PassWall2 如何结合延迟选择最优节点？
PassWall2 定期测速节点延迟，并结合策略组自动选择延迟最低的节点。无需手动操作即可保证网页、视频和游戏流量快速稳定访问，提高整体网络体验。
Sing-box 的 JSON 配置文件常见语法错误如何快速定位？
通过 JSON 校验工具或编辑器（例如 VSCode）打开配置文件，查看语法高亮与错误提示；也可运行 sing-box 并观察控制台错误信息，错误通常会指出第几行或哪个字段不合法以便修正。
Clash 策略组如何提升节点切换效率？
通过策略组可设置节点优先级和分流规则，结合延迟检测功能，Clash 自动选择延迟低、稳定性高的节点处理不同流量，提高网页、视频和游戏访问速度及整体网络体验。
Shadowrocket 可以在哪些平台使用？
Shadowrocket 是一款 iOS 平台专用代理客户端，支持多种协议如 Vmess、Shadowsocks、Trojan 等。用户可通过导入订阅链接快速配置节点，配合规则分流功能实现精准网络代理。由于其强大功能，它是 iPhone 用户常用的科学上网工具。
SSR 节点和 V2Ray 节点可以混合使用吗？
可以，但要使用支持多协议的客户端，如 Clash、Sing-box、Mihomo 等。这类客户端能在同一配置文件中加载不同协议的节点，通过规则分流让 SSR 与 V2Ray 同时工作。
Shadowrocket 的分流规则如何实现广告拦截？
通过配置规则，将广告域名或请求路径匹配到黑名单策略，使请求直接阻断或重定向到本地地址。这样用户浏览网页或使用应用时可自动屏蔽大部分广告内容。
Nekoray 节点订阅失败如何处理？
首先确认订阅 URL 是否有效，并确保网络连接正常。可手动刷新或重新添加订阅链接，保证节点列表最新。结合策略组使用，确保稳定科学上网体验。
